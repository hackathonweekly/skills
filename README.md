# HackathonWeekly Skills

A collection of AI skills for the HackathonWeekly community.

## Installation

```bash
npx skills add hackathonweekly/skills
```

## Available Skills

### Poster Generator

Generate beautiful event posters in three distinct styles.

**Usage**: Ask Claude to create a poster for your event, and it will generate 3 design options for you to choose from.

**Styles**:
- Black-White Minimalist: Clean, brutalist aesthetic
- Brutalist Modern: Bold, experimental layout

**Output**: HTML previews + PNG files in poster (1080x1440) and WeChat header (900x383) sizes.

See [poster-generator/poster-generator.md](poster-generator/poster-generator.md) for details.
